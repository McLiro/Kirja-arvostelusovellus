# Kirja-arvostelusovellus

Sovelluksessa voi julkaista kirja-arvostelun, jonka muut käyttäjät voivat lukea ja kommentoida.

Sovelluksen ominaisuuksia:
  - Käyttäjä voi kirjautua sisään ja ulos tai luoda uuden käyttäjän.
  - Käyttäjä näkee etusivulla listan muista kirja-arvosteluista ja arvostelijan antaman arvosanan.
  - Käyttäjä voi luoda uuden kirja-arvostelun antamalla kirjan nimen, arvosanan ja arvostelun.
  - Käyttäjä voi kommentoida olemassa olevaan arvosteluun.
  - Käyttäjä voi etsiä arvosteluja kirjan nimen perusteella.
  - Ylläpitäjä voi poistaa arvosteluja ja kommentteja.
